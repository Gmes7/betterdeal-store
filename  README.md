# BetterDeal E-Commerce Store

A complete e-commerce solution with business management features.

## Features
- Product catalog and shopping cart
- Order management
- Business dashboard with financial reports
- Employee management
- Expense tracking
- Mobile responsive design

## Installation
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
